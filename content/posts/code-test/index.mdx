# 程式碼測試

這是一個 TypeScript 範例：

```typescript
interface User {
  id: number;
  name: string;
}

const getUser = (id: number): User => {
  return {
    id,
    name: "User " + id,
  };
};

console.log(getUser(1));
```

這是一個 Python 範例：

```python
def fib(n):
    if n <= 1:
        return n
    return fib(n-1) + fib(n-2)

print([fib(i) for i in range(10)])
```

## 內嵌程式碼

也可以使用 `console.log("hello")` 這樣的方式。
