VS Code 有非常多快捷鍵，我整理了一些較常用的，可以加速工作效率。

## 編輯程式

| 快捷鍵                | 功能                                               |
| --------------------- | -------------------------------------------------- |
| **Ctrl+/**            | 註解 / 取消註解（可以一次註解多行，超級好用）      |
| **Shift+Alt+Up/Down** | 複製行至上/下                                      |
| **Alt+Up/Down**       | 移動行                                             |
| **Ctrl+D**            | 將下一個搜尋符合項新增至選擇範圍（大量改變數名稱） |
| Ctrl+X                | 刪除行（其實是剪下，但拿來當刪除很方便）           |
| Ctrl+Shift+K          | 刪除行                                             |
| Ctrl+K -> Ctrl+C      | 註解                                               |
| Ctrl+K -> Ctrl+U      | 取消註解                                           |
| Ctrl+F                | 尋找                                               |

> **Tips:** `Ctrl+K`是 VS Code 的前置快捷鍵，先按下之後再按其他快捷鍵會有不 同效果（可能因為快捷鍵太多所以才有這個功能）

## 版面

| 快捷鍵         | 功能                                         |
| -------------- | -------------------------------------------- |
| **Ctrl+Enter** | 創建新行（游標在中間也會直接跳到新行）       |
| **Ctrl+`**     | 開啟 / 關閉終端機                            |
| Ctrl+Up/Down   | 移動頁面                                     |
| Ctrl+K -> W    | 關閉所有分頁                                 |
| Ctrl+K -> U    | 關閉已儲存分頁                               |
| Alt+Left/Right | 切換分頁                                     |
| Ctrl+P         | 跳轉到文件（如果專案很多文件就可以直接搜尋） |
| Ctrl+Shift+P   | 開啟 VS Code 命令列                          |

## 執行程式 (Code Runner)

| 快捷鍵         | 功能                     |
| -------------- | ------------------------ |
| Ctrl+Alt+N     | 執行程式                 |
| Ctrl+Alt+M     | 停止程式                 |
| **Ctrl+Alt+K** | 執行自定義命令（超實用） |
